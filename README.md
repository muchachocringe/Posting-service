Сервис для отслеживания посылок который хранит и менет данные в БД
Функцианал:
- регистрация посылки,
- получение списка посылок клиента,
- изменение статуса посылки,
- изменение адреса доставки,
- удаление посылки.

- В качестве СУБД используется SQLite
- В файле parcel_test.go находятся тесты для функций из parcel.go
